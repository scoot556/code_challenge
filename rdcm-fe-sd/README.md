  ## Installing/Running Exercise
  #### Requirements
- Node version: v18.18.2
- NPM version: 9.8.1

#### Installation

 1. Clone repository
 2. Change directory to this folder
 3. Run `npm install`
 4. Run `npm run dev`
 5. Go to `http://localhost:3000`
